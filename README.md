# Sales-Superstore-Dashboard
This project is a Sales Performance Dashboard created in Power BI to analyze sales, profit, customer segments, and regional performance. It provides interactive visualizations and KPI tracking to help understand business trends and generate meaningful insights from sales data.

# Key Features
- Sales and Profit Analysis
- Regional Performance Tracking
- Category & Sub-Category Analysis
- Monthly Sales Trend Monitoring
- Customer Segment Insights
- KPI Dashboard Reporting
- Interactive Filters and Visualizations


# Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Modeling
- Data Visualization
